## 1. Описание актива и инструмента

Актив: акции ПАО «Яндекс» (YDEX) на Московской бирже. Высоколиквидный инструмент с выраженными трендовыми движениями. Период тестирования: 2024–2026. Источник данных: Finam API. Таймфреймы: 1 ч и 15 мин.

Индикаторы: EMA(20), EMA(60), EMA(100), дневной VWAP, ATR(14), RSI(14), ADX(14). Вход: EMA(20) > EMA(60) и цена > VWAP для лонга; обратные условия для шорта. Выход: TP/SL в единицах ATR, breakeven после движения в прибыль, трейлинг стоп.

---

## 2. Метод 1 — изолированность входа и выхода

Базовый вариант (выход через N баров, кумулятивный VWAP): система убыточна при всех N (10, 20, 30, 40 баров). PF < 1.

Улучшенный вариант: дневной VWAP, фильтр ATR при входе, стоп 1.5·ATR. Результаты:
- выход через 40 баров: P = 796.30, PF = 1.27, 93 сделки, MIDD = 622
- выход через 10 баров: P = 771.40, PF = 1.40, 101 сделка

---

## 3. Метод 2 — визуально-графический анализ

Решётка периодов EMA (fast: 10–30, slow: 40–80) на улучшенной схеме 1h с фильтрами (тренд EMA100, ADX≥18, RSI, объём). Прибыльными оказались все 25 комбинаций. Лучшая зона: EMA 20/40 — P ≈ 830, PF ≈ 1.49, RF ≈ 4.0. Устойчивость подтверждена: прибыльна не одна точка, а область параметров.

## 5. Оценка качества ТС

По лучшему сетапу (1h, trend_adx_rsi): P = 541.84, PF = 1.39, RF = 2.29, MIDD = 236.61. Доля прибыльных ≈ 43%. Количество сделок 107 — достаточно для предварительных выводов. RF > 2 свидетельствует об устойчивости к просадкам.

## 6. Параметры финальной МТС

- Таймфрейм: 1h
- EMA 20/40 (или 20/60)
- Дневной VWAP
- Фильтр тренда: close > EMA(100) для лонга, close < EMA(100) для шорта
- ADX ≥ 18, RSI ≥ 52 для лонга и RSI ≤ 48 для шорта
- Объём ≥ 0.9·vol_ma
- TP = 1.5·ATR, SL = 1.5·ATR
- Breakeven после +1·ATR, трейлинг после +1.5·ATR (фиксация 0.5·ATR)
- Сессия: 10:00–17:00
